# 🚀 CPU Scheduling Algorithm Simulator (Shell Project)

## 📌 Overview
This project is a CPU Scheduling Algorithm Simulator implemented in Bash Shell Script.

### Algorithms Included
- First Come First Serve (FCFS)
- Shortest Job First (SJF)
- Priority Scheduling

## ⚙️ How to Run

1. Give permission:
   chmod +x scheduler.sh

2. Run:
   ./scheduler.sh

## 📊 Output
Displays:
- Waiting Time
- Turnaround Time
- Average Waiting Time
- Average Turnaround Time

## 🛠 Requirements
- Linux / macOS / WSL
- Bash
- bc (for floating point calculations)
